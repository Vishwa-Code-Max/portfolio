## About

This is my personal developer portfolio — a fast, animated, and interactive web experience built to showcase my projects, skills, and journey as a MERN Stack Developer.

> Designed and developed fully from scratch. No templates.

---

## Tech Stack

| Layer | Tech |
|---|---|
| Framework | React.js + Vite |
| Styling | Tailwind CSS |
| Animations | GSAP (ScrollTrigger, Timelines) |
| 3D / Visual | Three.js + Lottie |
| Deployment | Netlify |

---

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Vishwa-Code-Max/portfolio.git

# Navigate into the project
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## Build for Production

```bash
npm run build
```

Output will be generated in the `dist/` folder.
